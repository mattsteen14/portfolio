Immature Greeting Generator

Details

When picking your child up from school do you ever struggle to respond to your childs immature and silly words?
Well, struggle no longer. Simply run this code for a preloaded greeting that will make you a hit at the playground.
I created this code for the previous portfolio project on Javascript syntax called Mixed Messages. See in the GitHub link below.
There are a number of subcategorised arrays nested inside of the immatureChoices object.
I created a generateRandomNumber helper function which I then call within a function to iterate through the arrays to push the relevant strings based on gender set in the parameter. The functions returns the joined string.
